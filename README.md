# Personal Portfolio using Frontend Web Development
## ✨ Project Overview
- This is a responsive Personal Portfolio Website developed using HTML5 and CSS3 to showcase my skills, projects, certifications, and professional profile.
- The website serves as an online portfolio where visitors can learn about me, explore my work, and access my resume.
- The portfolio features a clean and modern user interface with smooth navigation, responsive layouts, interactive project cards, and direct links to my GitHub repositories, Figma prototypes, LinkedIn profile, and resume.
- It is designed to provide a simple yet engaging user experience across different devices.
## 🎯 Objectives
- The primary objective of this project is to build a professional online portfolio that showcases my frontend development and UI/UX design skills while providing recruiters and visitors with easy access to my projects, certifications, and resume.
## ⭐️ Features
- Responsive and modern portfolio design
- Fixed navigation bar with smooth scrolling
- About Me section introducing my background
- Skills section highlighting technical and design skills
- Projects section with project descriptions and live demo links
- Certificates section showcasing achievements
- Resume download/view option
- Social media links (GitHub & LinkedIn)
- Attractive UI with hover effects and gradient styling
## 🛠️ Technologies Used
- HTML5
- CSS3
- Git & GitHub
- GitHub Pages (Deployment)
